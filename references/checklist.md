# Prompt Forge Checklist

交付前快速检查：

- [ ] 我知道目标模型或已显式声明默认假设。
- [ ] prompt 的主要产物是清晰可复制的，不是方法论散文。
- [ ] 角色、目标、上下文处理、工作流、约束、输出格式都已定义。
- [ ] 复杂任务已补 few-shot，且至少有一个边缘场景。
- [ ] 已加入负向约束，尤其是禁止臆造和禁止忽略用户材料。
- [ ] 如果涉及文件、图片、长文本，已写清处理优先级。
- [ ] 如果目标是 Gemini，已使用 XML 或至少按 Gemini 原生特性设计。
- [ ] 如果任务是优化旧 prompt，已做红队漏洞扫描。
- [ ] 如果用户要求“只给 prompt”，最终输出不会夹带多余解释。
- [ ] 该 prompt 没有和 requirement 文档、task plan、实现方案混成一团。
